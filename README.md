# 🧠 VitaAI — Your Health Companion ❤️

VitaAI is an AI-powered healthcare and mental wellness platform designed to provide **24/7 intelligent health support**. It combines AI chat, mood tracking, counselling, and health monitoring — all in one place.

---

## 🚀 Features

### 🤖 AI Health Assistant
- Chat with an AI for health-related questions
- Get guidance on symptoms, medications, and wellness
- Available 24/7

### 🌡️ Mood Tracker
- Track your emotional wellbeing
- Get personalized wellness insights
- Detect early mental health patterns

### 💬 Online Counselling
- Book sessions with professional counsellors
- Multiple session types (video, voice, chat)
- Real-time availability tracking

### 🎯 Support System
- Submit health queries
- Track ticket status (Open / In Progress / Resolved)
- Get expert responses

### 📊 Vitals Monitor
- Track:
  - Heart rate ❤️
  - Blood oxygen 💧
  - Body temperature 🌡️
  - Sleep quality 😴
- Visual health dashboards

### 💊 Smart Reminders
- Medication reminders
- Health habit tracking

- JavaScript (Vanilla JS)
